# Explainable AI (XAI) & Model Interpretability Analysis

A specialized project dedicated to solving the "black-box" problem in machine learning. This analysis evaluates high-performance models (XGBoost) for ad-click prediction and retail profit analysis, ensuring that model decisions are transparent, ethical, and business-validated.

1. Interpretability Framework: Implemented SHAP (SHapley Additive exPlanations) and Counterfactual Analysis to decode complex feature interactions, such as the "Dark Forest" interaction effect between income and store location.

2. Global & Local Explanations: Developed Surrogate Models (Decision Trees and Linear Regressors) to approximate complex model behavior into human-readable rules for stakeholders.

3. Model Reliability: Integrated CUSUM and Page-Hinkley drift detection algorithms to monitor streaming data, ensuring the model remains accurate and reliable over time by identifying statistical shifts in real-world data.
